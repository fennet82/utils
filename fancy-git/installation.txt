installation guide
------------------

NOTE: will only work for bash terminals!! (obviously will not work for windowsw)

1. pull the repository and decrypt the files with the decryption extractor.sh file
2. important youll need to go in the explorer to the ./.fancy-git/ and then go to fonts and install each by clicking them 
3. chmod +x and then run the file install.sh in ./.fancy-git/ 
4. and configure the windows terminal to this font: "SourceCodePro+Powerline+Awesome Regular" (settings -> distrobution -> appearnce -> font)
5. close the terminal (try multiple times)
